Implemented both proximal policy optimization (PPO) and constrained PPO (CPPO) to compare the number of safety violations and learning rate differences between the algorithms.

Python PyTorch Control Theory Reinforcement Learning

Used a Lagrangian-based constrain on typical PPO implementations for the algorithm. The agent operated in a Pybullet gymnasium that contained an obstacle the agent should avoid when navigating to the goal position.

Challenge:Traditional model-free RL algorithms don't provide formal safety guarantees for drone navigation and can exhibit harmful behaviors during the learning phase.
